РУДАНСЬКИЙ СТЕПАН
«Голубонько-дівчинонько…» 


Голубонько-дівчинонько,
&nbsp; &nbsp; &nbsp; Зіронько моя!
Не питайся, моя мила,
&nbsp; &nbsp; &nbsp; Чого смутен я!
Живо-живо того смутку
&nbsp; &nbsp; &nbsp; І ти не минеш!..
Отоді ти, моя мила,
&nbsp; &nbsp; &nbsp; Мене спом’янеш…

Голубонько-дівчинонько,
&nbsp; &nbsp; &nbsp; Мій рожевий цвіт!
Ти весела, світом рада,
&nbsp; &nbsp; &nbsp; Тобі милий світ!..
Живо-живо світом знудиш,
&nbsp; &nbsp; &nbsp; Тяженько здихнеш…
Отоді ти, моя мила,
&nbsp; &nbsp; &nbsp; Мене спом’янеш!..

Гудуть пчоли коло тебе,
&nbsp; &nbsp; &nbsp; Ти, як лист, дрижиш…
Ти їх гониш, не одгониш
&nbsp; &nbsp; &nbsp; І сама біжиш…
Живо-живо під ті співи,
&nbsp; &nbsp; &nbsp; Як дитя, заснеш…
Отоді ти, моя мила,
&nbsp; &nbsp; &nbsp; Мене спом’янеш…

Ти затопиш очі в очі —
&nbsp; &nbsp; &nbsp; Очі не схибнуть!
Тобі дивно, що чужії
&nbsp; &nbsp; &nbsp; Від твоїх змигнуть…
Живо-живо сама глянеш —
&nbsp; &nbsp; &nbsp; І сама змигнеш.
Отоді ти, моя мила,
&nbsp; &nbsp; &nbsp; Мене спом’янеш…

Ти вітаєш, обіймаєш
&nbsp; &nbsp; &nbsp; Своїх і чужих!
Як дитина, ти, дівчино,
&nbsp; &nbsp; &nbsp; Ластишся до них…
Живо-живо од чужого
&nbsp; &nbsp; &nbsp; Руки одвинеш…
Отоді ти, моя мила,
&nbsp; &nbsp; &nbsp; Мене спом’янеш…

Прийде ранок — ти весела,
&nbsp; &nbsp; &nbsp; Других веселиш…
Прийде нічка — ти в постелі,
&nbsp; &nbsp; &nbsp; Як голубка, спиш…
Живо-живо цілі ночки
&nbsp; &nbsp; &nbsp; Оком не скинеш…
Отоді ти, моя мила,
&nbsp; &nbsp; &nbsp; Мене спом’янеш…
